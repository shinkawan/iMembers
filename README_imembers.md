# iMembers プラグイン開発完了レポート

このリポジトリには、高機能メンバーシップ管理プラグイン「iMembers」のすべてのソースコードが含まれています。

## 実装済みの機能

### 1. コンテンツ制限
- ページ単位の「会員限定」設定。
- カテゴリー・タグ単位の一括制限。
- `[members_only]` ショートコードによるチラ見せ機能。
- `[imembers_download]` による会員限定ファイル配布。

### 2. 認証システム
- 認証コード（OTP）によるパスワードレスログイン。
- SNSログイン（LINE/Google）のルーティング基盤。
- 30日間のログインセッション保持。

### 3. ユーザーアクティビティ
- Ajaxによる「お気に入り」登録。
- 直近20件の閲覧履歴。
- 専用DBテーブル（`wp_imembers_activity`）による高速なデータ処理。

### 4. Stripeサブスクリプション
- プラン契約・管理ポータル連携。
- Webhookによる支払い失敗時の自動権限剥奪。

## 開発構造
- `includes/`: クラスベースの機能実装。
- `templates/`: UIテンプレートの分離（保守性重視）。
- `assets/`: フロントエンド用スクリプト。
